交错矩阵星链定位由两部分构成：星链（通信）和定位，通过小型路侧基站可以无死角覆盖主要区域，，而且路侧设备具备一定算力，有着多种外设接口形式和通信形式，可以提供统一的标准接入不同厂家的传感器设备，该基站既能实现无人驾驶的高精度定位，还能对人员物资进行定位，也能把接入的传感器设备采集的数据通过一张星链网传输出去，还能够让井下覆盖网络，实现手机通过每个基站的WIFI进行与井上人员的沟通。

2、详细技术说明
2.1交错矩阵星链
交错矩阵星链（Interleave Division Multiple Access，IDMA）是一种多址访问技术，它可以用于超宽带（Ultra-Wideband，UWB）通信系统中，以实现多用户间的高效通信。下面是交错矩阵星链定位系统的工作原理：
用户分配：在UWB系统中，首先对用户进行分配。每个用户都有一个唯一的用户码，这个码被用来区分不同的用户。在IDMA中，用户之间的码是通过一种交错的方式分配的，以增加系统的容量和效率。
交错矩阵：交错矩阵是IDMA系统的核心概念。它是一个矩阵，其中每行表示一个用户的传输数据，而每列表示一个时间槽。用户的数据被编码为矩阵中的元素，并根据其用户码的位置放置在不同的列中。这种交错矩阵的设计使得多个用户的数据可以在相同时间内传输，而不会发生冲突。
发送端：在发送端，用户的数据被编码并映射到交错矩阵的相应位置。然后，UWB信号被产生，包括用户数据、同步信号以及其他必要的控制信息。这些信号通过UWB天线传输。
接收端：在接收端，UWB信号经过接收天线接收后，被分离成不同用户的信号。然后，通过解交错矩阵，可以还原各用户的数据。接收端使用用户的唯一用户码来确定正确的数据位置，以解码数据。
多用户分离：由于每个用户的数据都以不同的方式映射到交错矩阵中，接收端能够分离和解码多个用户的数据，而不会发生干扰。
反馈和重传：如果数据包丢失或受到干扰，系统可以使用反馈机制请求重传，以确保数据的可靠传输。
总的来说，交错矩阵星链定位系统通过在时间和空间上交错多个用户的数据，以及利用唯一的用户码来进行多用户分离，实现了高效的多址访问和可靠的数据传输，适用于UWB通信系统中的多用户环境。这种技术有助于提高UWB系统的容量和性能。
2.2煤矿用星链系统
煤矿用星链系统是一种用于提供通信、定位和监测服务的无线通信技术，特别适用于煤矿等地下环境。其工作原理如下：
网络拓扑：在煤矿环境中，通常采用星型网络拓扑，其中一个基站或控制节点位于地下，而其他设备，如矿工身上携带的设备，作为终端节点连接到基站。这种拓扑结构有助于简化通信管理和维护。
UWB技术：UWB技术采用了非常宽的频带，允许传输非常短脉冲的信号。这些短脉冲信号在地下环境中具有较好的穿透能力和抗干扰性，有助于提供可靠的通信。
位置定位：星链定位系统可以用于高精度的位置定位，对于煤矿来说尤为重要。基站和终端节点之间的距离可以通过测量信号的到达时间来确定，从而实现准确的室内或地下位置定位。
通信和数据传输：基站通过UWB信号向终端节点广播数据或命令，终端节点接收并处理这些数据，然后返回响应。这种通信方式可以用于煤矿中的紧急通信、监测数据传输以及矿工之间的语音和文本通信。
多径传播和抗干扰：地下环境中通常存在多条信号传播路径，因此UWB系统需要处理多径传播问题，以确保数据的准确传输。同时，UWB技术具有较强的抗干扰性，能够在有噪声干扰的环境中工作。
安全性：由于煤矿环境的特殊性，通信系统需要提供高度的安全性，以确保通信数据不会被未经授权的人访问。因此，加密和身份验证机制通常被应用在煤矿UWB系统中。总的来说，煤矿用星链系统通过使用UWB技术实现地下煤矿的通信、定位和监测需求。这种系统具有高可靠性、高精度位置定位能力和抗干扰性，有助于提高煤矿工作的安全性和效率。
2.3 煤矿用星链系统算法（DS-TWR）
煤矿用星链系统采用的DS-TWR（Time of Flight with Ranging）算法，该算法是一种用于测量无线信号传播时间的定位算法，被广泛应用于煤矿井下星链定位系统（如图5）。






图5 DS-TWR工作原理
其工作原理如下：
信号发送：在星链定位系统中，一个设备充当星链基站节点，发送一个短脉冲的无线信号。这个信号以高速传播，以移动标签模块为目标。
信号接收：移动标签模块接收到发送的信号。设备内部的UWB接收器会记录接收到信号的时间戳，即信号的到达时间。
时间差测量：移动标签会记录接收到信号的时间戳，并将其发送回星链基站节点。星链基站节点也会记录发送信号的时间戳。因此，星链基站节点和移动标签都有了两个时间戳：一个是信号发送的时间戳，一个是信号接收的时间戳。
计算传播时间：通过计算星链基站节点和移动标签之间信号传播的时间差，即两个时间戳之间的差值，可以获得信号的传播时间，也称为"飞行时间"。这个时间差告诉系统信号从星链基站节点传播到移动标签所需的时间。
距离计算：通过测得的信号传播时间，可以使用光速或电磁波传播速度来计算信号传播的距离。距离计算通常采用以下公式：距离 = 传播时间×速度。在星链系统中，由于其极短的脉冲，精确度很高，d=[（t1-tt1)+(t2-tt2)]c/4，其中d为星链基站节点到移动标签终端距离，c为信号在介质中的传播速度。
多个星链基站节点：在实际应用中，通常会使用多个星链基站节点，每个星链基站节点都发送信号并记录传播时间。通过与多个星链基站节点的信号传播时间差，可以进行三角定位或多边定位来确定移动标签的位置。

TWR算法与传统的TOA和TDOA算法相比，具有以下优势：
（1）高精度：TWR算法通常能够提供非常高的定位精度。这是因为它通过测量信号传播时间差来计算距离，而UWB信号的短脉冲可以提供极高的时间分辨率。因此，TWR算法通常能够实现亚米级别的高精度定位，适用于需要高度准确性的应用，如煤矿井下无人驾驶车辆的定位。
（2）无需基础设施：TWR算法不需要昂贵或复杂的基础设施，只需使用星链基站节点设备和移动标签，因此相对成本较低。这对于在煤矿等环境中进行部署和维护非常重要。
（3）实时性：TWR算法提供实时位置信息，使其适用于需要快速决策和操作的应用，如实时监控和紧急响应。
（4）抗干扰：由于UWB信号的宽带特性，TWR算法在一些干扰环境中表现出良好的抗干扰能力。它能够处理信号反射和多径效应，提供稳定的测量结果。
（5）可扩展性：TWR算法可以扩展到多个星链基站节点，允许系统实现更高的精度和覆盖范围。这使其适用于不同大小的煤矿井下区域。
（6）精准度不受环境影响：DS-TWR算法不依赖于环境条件，如温度或湿度，因此在不同的煤矿井下环境中也能提供一致的精准度。

2.4星链定位系统部署
项目首要任务是在煤矿井下部署和搭建“基于交错矩阵式星链定位系统”。这包括选择和部署星链基站节点，这些节点将被放置在矿井的关键位置，以用于位置定位和通信。此外，项目还将配置车载移动标签，这些标签将附在无人驾驶车辆上，以便跟踪它们的位置。星链路测端用于测量星链UWB信号的传播特性，以帮助确定节点位置。最后，星链节点校准仪将用于确保节点之间的准确互动。

一旦部署，需要对系统进行校准和配置，以确保各个组件协同工作并提供精确的定位信息。这可能涉及星链节点的位置校准，以确保它们的位置准确；还包括频率设置和信号强度调整，以优化系统性能。
2.6数据收集和处理

2.7安全合规
项目将追求取得安标（煤矿安全标准）认证，以确保系统满足相关法规和标准。这是确保工作环境安全性的重要步骤，特别是在煤矿井下环境中。
2.8多传感器融合技术
在项目中，星链定位技术将与其他传感器（如激光雷达、摄像头和惯性导航）融合，以提供更全面的位置定位和监测。
2.9无人驾驶车辆控制
项目的一部分将涉及与无人驾驶车辆的控制系统集成，以实现精确的车辆定位和自主行驶。这将包括车辆的自主导航、障碍物检测和路径规划等功能。
2.10实验和测试
项目将进行大量实验和测试，以验证系统的性能和可靠性。这可能包括在模拟煤矿井下环境中进行实地测试，以模拟真实工作环境，并确保系统能够在不同情况下正常运行。
2.11数据分析和决策支持

六、采购要求
1.供货方应对货物的质量、规格、性能进行检验，出具合格质量检验证明文件。
2.供货时、所有货物应分类，包装整齐，包装上应有明显的所装货物的型号、规格、数量标记，集中装箱的应有装箱单，方便采购方核对，货物包装散乱的采购方有权拒收。
3.货物供货时应提供发货单纸质版及电子版、装箱单和产品质量证明文件，方便供货方进行验收。
4.货物到采购方指定场地后，原则上由采购方、供货方双方应进行现场交接验收，因特殊原因，供货方不能到厂时，采购方可单独验收，验收结果供货方须予以承认。
5.供货方应提供对所供货物使用、装配、技术、售后服务支持，复杂设备应提供设备安装、使用、操作、维护说明书，如采购方认为有必要，应派遣专门人员对采购方进行培训。
6.在交货前，供货方应完成对货物质量、规格、性能等内容精确全面的检验，对检验过的符合规定的出具质量检验证明。
7.采购方有派人员进行过程监督的权利，供货方为采购方监理提供食宿等便利。
8.如果因供货方所供货物质量原因，造成采购方人员受伤或设备损坏，采购方将对供货方提出索赔。
七、技术与培训服务
1、应答人提供的资料使用国际法定计量单位，以中文为准。图纸、资料除提供书面文件外，还应提供电子版文件。图纸为AutoCAD格式，文本文件为Word/Excel格式。
2、应答人负责提供所有有关技术资料，包含但不限于以下内容：
（1）提供给产品说明书、使用手册等技术文件。
（2）提供产品名称、版本号、软件著作权人等。
（3）提供产品的技术文档和应用开发与定制部分的源代码，并对采购人进行技术交底，以供采购人今后进行开发和系统维护使用。
3、技术资料与设备同属合同供货范围，如不能按照上述条款交货，应按照推迟合同交货期执行。
4、应答人派出有技术、有能力胜任的须具有相应的资质服务工程师到现场，提供有关操作、使用培训人员的服务，应答人服务工程师的主要责任与任务如下：
（1）给采购人人员提供完整的或系统的技术指导；
（2）现场培训采购人人员；
（3）质保期内技术服务。
5、由应答人的技术服务人员给予采购人使用人员提供软件模块功能介绍、讲课与培训；详细解释技术文件、和操作手册以及安全运行和相关的预防措施等；回答和解决采购人人员提出的技术问题。应答人技术人员的指导必须是正确的，如果出现由于非正确技术指导而造成的损失，应答人将自出资金更换和或补偿损失部分。
6、应答人应提供用于采购人自行培训人员需要使用的相关培训软件（动画形式）、教学模型（透明可视）。

1、应答人保证所提供的产品满足技术条件、性能及功能的要求，符合国家标准。
2、应答人应向采购人提供有关新技术、交流信息和咨询服务。
3、应答人应免费提供技术培训，长期向采购人提供维护保养技术。
4、应答人对采购人及采购人指定的第三方开放所有接口、数据，并全力配合采购人的智慧港口建设工作。

8、工期：合同签订后24个月内完成研发并投入试运行。
